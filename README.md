# Titanic-Survival-Prediction

## Objective
Build a machine learning model to predict whether a passenger survived the Titanic disaster based on passenger details.

## Dataset
Kaggle Titanic Dataset - 891 passengers with features like age, gender, class, fare, etc.

## Tech Stack
- **Language**: Python
- **Libraries**: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn  
- **Platform**: Google Colab

## ML Model
- **Algorithm**: Random Forest Classifier
- **Accuracy**: 82%

## Workflow
1. Data Loading & Exploration
2. Handling Missing Values 
3. Encoding Categorical Variables
4. Feature Selection
5. Train-Test Split
6. Model Training & Evaluation

## Key Findings
- Women and children had higher survival rate
- 1st class passengers survived more than 3rd class
- Random Forest gave best accuracy of 82%

## How to Run
1. Open `Titanic_Survival_Prediction_Samruddhi.ipynb` in Colab
2. Run all cells
3. Check `Titanic_Survival_Prediction_Report_Samruddhi.pdf` for detailed analysis
